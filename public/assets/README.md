# Vidhi Page Assets

This folder contains all assets for the Vidhi page.

## Required Assets

### Hero Section
- `hero.mp4` - Full-screen background video (autoplay, muted, loop)
- `hero-poster.jpg` - Poster image for the video (shown before video loads)

### Services Carousel (7 images)
- `service-1.jpg` - Shaadi Planning
- `service-2.jpg` - Haldi Ceremony
- `service-3.jpg` - Sangeet Night
- `service-4.jpg` - Pre-Wedding Shoot
- `service-5.jpg` - Post-Wedding Shoot
- `service-6.jpg` - Destination Wedding
- `service-7.jpg` - Decor & Theme

### Signature Moments (9 images)
- `moment-1.jpg` through `moment-9.jpg` - Gallery images for signature moments section

### Testimonials (4 images)
- `testimonial-1.jpg` through `testimonial-4.jpg` - Circular portrait images for testimonials

## Image Specifications

- **Services images**: Recommended 400x300px or 16:9 aspect ratio
- **Moment images**: Recommended 600x600px (square) or 4:3 aspect ratio
- **Testimonial images**: Recommended 200x200px (square, will be displayed as circular)
- **Hero poster**: Recommended 1920x1080px or 16:9 aspect ratio
- **Hero video**: Recommended 1920x1080px, MP4 format, optimized for web

## Note

The code includes fallback placeholder images, so the page will work even if these assets are not yet available. Replace the placeholder URLs with actual images when ready.
